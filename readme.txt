15-soldiers_of_victory15.pdf - pdf документ, содержащий в себе данные для обработки

parsing.ipynb - парсер информации с документа, попутно её обрабатывающий

parsed_data.csv - результат парсинга программы parsing.ipynb

model.ipynb - модель обучения, которая должна была присутствовать, но из-за нехватки времени, идей и проблемы с форматом документа не была создана.

application.py - приложение с графическим интерфейсом, позволяющее открывать .csv файлы для просмотра и добавления новых данных

Для работы с application.py необходимо выбрать нужный нам набор данных (parsed_data.csv), после чего его можно просматривать и редактировать
